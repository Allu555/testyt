# YT Music Player 🎵

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/Allu555/ytplayer)

A premium, Spotify-inspired frontend YouTube Music Player wrapper with a beautiful "heartbeat" progress bar.

## 🚀 One-Click Deployment
1. Click the **Deploy to Netlify** button above.
2. Connect your GitHub account.
3. That's it! Netlify will automatically build and host your player.

## 🛠️ Local Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Start the backend: `python server.py`
3. Serve the frontend: `npx serve public`
